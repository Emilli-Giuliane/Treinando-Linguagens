Detalhes da oferta

Resumo

Este curso oferece uma base sólida em programação, lógica de programação e desenvolvimento de algoritmos. Os estudantes irão explorar os fundamentos essenciais da criação de software, desenvolvendo competências para resolver problemas computacionais de maneira estruturada e eficiente. O curso abrange conceitos teóricos e práticos, preparando os alunos para iniciar sua jornada no mundo da programação com a base de lógica adequada. Conceitos fundamentais de programação orientada a objetos ¿ classes, objetos, operações, mensagens, métodos; polimorfismo; identificação de objetos; abstrações, generalização, subclasses, super classes, instanciação; herança; encapsulamento; abstração de agregação; abstração de composição, construtores; introdução à linguagem java.
